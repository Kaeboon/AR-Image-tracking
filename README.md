# AR-Image-tracking
This is a project done under Tel-imaginarium workshop.
It is an marker-based AR application that can track fruit images on phone, and display their related colors. It can be used for teaching very young children who are learning the colour of fruits.

This project is build using Vuforia on Unity 2019.4.26f1. Three ImageTargets were used, For each ImageTarget, a sphere with the fruitâ€™s colour and a 3d text showing name of the colour. Packages used include ARFoundation, ARCore, ARKit.
